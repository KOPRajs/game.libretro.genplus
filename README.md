# Sega - MS/GG/MD/CD (Genesis Plus GX)

### Description

Genesis Plus GX is an enhanced port of Genesis Plus to the libretro API.

Features support for SG-1000, Mark-III, Master System, Game Gear, Mega Drive and Mega CD hardware.
